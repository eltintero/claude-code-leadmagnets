Describe a Task. Walk Away. Come Back to a Finished Deliverable.

Everything up to now has been conversational. You type, Claude responds, you refine. That's Chat mode.

Cowork mode is different. You describe a task, Claude plans its approach, executes it step by step, and delivers the finished result. It reads files, writes documents, searches your connected tools, and works through multi-step processes — all while you do other work.

The mental shift: Chat = conversation. Cowork = delegation.

Cowork has three pillars:

  • Plan — Claude outlines its approach before executing. You can review and adjust.
  • Execute — Claude works through the plan, using connected tools and files as needed.
  • Connect — Claude pulls from your connectors (Slack, Drive, etc.) during execution.

File tasks work on specific documents. Scheduled tasks run automatically — daily report compilations, weekly summaries, recurring data pulls.
When to use Cowork vs. Chat:
• Chat — quick questions, brainstorming, back-and-forth refinement
• Cowork — defined deliverables, multi-step tasks, anything you'd delegate to a junior team member

Today's Exercise • 20 minutes

Step 1: Start a Cowork session with a real task.

Pick something that has a clear deliverable. Ideas:

  • "Review this folder of meeting notes and create a summary document with action items, owners, and deadlines."
  • "Analyze our last 10 client proposals and create a template based on common patterns."
  • "Draft a quarterly review document pulling data from [connected tool]."

Step 2: Review Claude's plan before it executes. This is the key moment. Does the plan make sense? Missing any steps? Adjust now, not after.

Step 3: Let it work. Go do something else. Come back to the finished deliverable.

Step 4: Set up a scheduled task.

Think of something that happens weekly: a status summary, a report compilation, a data pull. Schedule it. Every scheduled task is a micro AI employee that shows up on time, every time.

Every scheduled task is a micro AI employee. Now multiply that across every department: sales gets automated pipeline reports, ops gets daily project summaries, finance gets weekly expense digests. That's what AI implementation looks like at scale — not one person using Claude well, but an entire organization with coordinated AI agents running in the background.
